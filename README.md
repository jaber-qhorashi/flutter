# Todo App

Eine einfache Flutter-Anwendung zum Verwalten von Aufgaben. Nutzer:innen können neue Todos anlegen, bearbeiten, abhaken und löschen sowie zwischen verschiedenen Filtern (alle, aktiv, erledigt) wechseln.

## Voraussetzungen

- [Flutter](https://flutter.dev) 3.16 oder neuer

## Entwicklung starten

```bash
flutter pub get
flutter run
```

Die App nutzt ausschließlich lokale Zustandsverwaltung und speichert Daten nicht dauerhaft. Sie eignet sich als Ausgangspunkt für Experimente mit Persistenz oder komplexerer State-Management-Lösungen.
